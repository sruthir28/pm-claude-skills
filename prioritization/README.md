## When to Use Each Framework

| Your Situation | Use This Framework |
|----------------|-------------------|
| Have quantitative data | RICE Scoring |
| Need quick sprint decision | Impact/Effort Matrix |
| High technical complexity | Value vs. Complexity |
| Custom business criteria | Weighted Scoring |
